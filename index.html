Assembler
September 2018 to November 2018

  • Technologies Used: C++, Assembly Language
  • Involved understanding assembly language and low-level programming.
  • The assembler consisted of two passes. Pass one parsed symbols from an assembly file and opcodes from a data file. After some validation, both were loaded into their respective tables and were used to increment the program location counter. Pass two generated object code with the data compiled from pass one, then created the relevant records. The result was then output as an object file.
